# react-native-chime-sdk

Amazon Chime SDK for react native

## Installation

```sh
npm install react-native-chime-sdk
```

## Usage

```js
import ChimeSdk from "react-native-chime-sdk";

// ...

const result = await ChimeSdk.multiply(3, 7);
```
